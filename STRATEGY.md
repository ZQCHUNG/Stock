# 交易策略說明

## 策略 v1（當前版本）— 六指標加權評分

### 概述
使用六大技術指標各自獨立評分（-1 ~ +1），加權後得到綜合分數，超過閾值觸發買賣。

### 指標與評分邏輯

#### 1. MA 移動平均線（權重 20%）
- MA5 > MA20：+0.5（短期趨勢向上）
- MA20 > MA60：+0.5（中長期趨勢向上）
- 反之各 -0.5
- **多頭排列（MA5>MA20>MA60）= +1，空頭排列 = -1**

#### 2. RSI 相對強弱指標（權重 20%）
- RSI < 30：+1（超賣反彈機會）
- RSI 30~50：線性插值 +1 → 0
- RSI 50~70：線性插值 0 → -1
- RSI > 70：-1（超買回檔風險）

#### 3. MACD（權重 20%）
- MACD > Signal 線：+0.5
- MACD 柱狀體 > 0：+0.5
- 反之各 -0.5

#### 4. KD 隨機指標（權重 20%）
- K > D：+0.5（黃金交叉）
- K < 20：+0.5（超賣區）
- K < D：-0.5（死亡交叉）
- K > 80：-0.5（超買區）

#### 5. 布林通道（權重 10%）
- 股價位置 = (收盤 - 下軌) / (上軌 - 下軌)
- 分數 = 1 - 2 * 位置（靠下軌偏多，靠上軌偏空）

#### 6. 成交量（權重 10%）
- 量增價漲：+0.5（健康上漲）
- 量縮價跌：+0.3（洗盤結束可能）
- 量增價跌：-1.0（出貨訊號）
- 量縮價漲：0.0（動能不足）

### 訊號產生
- 綜合評分 > 0.3 → **BUY（買入）**
- 綜合評分 < -0.3 → **SELL（賣出）**
- 其他 → **HOLD（持有/觀望）**

### 交易規則
- 買入：用全部可用資金，以「張」為單位（1000 股）
- 賣出：全部出清
- 一次只持有一檔（單一股票回測）

### 費用
- 手續費：0.1425%（買賣各收一次）
- 交易稅：0.3%（賣出時收）

---

## v1 已知問題

1. **沒有停損** — 虧損不設限，等到賣出訊號才出場
2. **沒有停利** — 不鎖利潤，大漲後可能反轉倒賠
3. **勝率偏低** — 指標交叉產生大量假訊號
4. **全押策略** — 每次 all-in 沒有資金管理
5. **無趨勢過濾** — 盤整期也頻繁進出，白交手續費

## 策略 v2（改進版）— 待實作

### 改進方向
1. **加入停損停利**
   - 停損：買入後跌超過 -7% 強制出場
   - 移動停利：從最高點回落 -5% 出場（鎖利潤）

2. **訊號確認機制**
   - 買入需連續 2 天評分 > 閾值才進場（過濾假訊號）
   - 成交量確認：買入當天量需 > 5 日均量

3. **趨勢過濾**
   - 只在 MA20 > MA60（中期多頭）時才做多
   - MA20 < MA60 時不進場，避免逆勢交易

4. **資金管理**
   - 單筆不超過總資金 50%
   - 依訊號強度調整部位大小
